Logitech solutions: MegaStore Global

¿Que hace y como?
Soluciona y gestiona de manera eficiente y optimizada la infraestructura, procesamiento y almacenamiento de datos del gigante de suministros ""MegaStore Global" por medio de una base de datos 
normalizada y estructura mediante PostgrSQL y DBeaver, además de una API robusta con Express, la cúal se encarga de las validaciones de acuerdo a la lógica de negocio y su respectivos CRUD.
